# WebApp - CRM

WebApp feito em angular 21 consumindo uma API feita em .NET 8.]
A idéia desse sistema é criar um CRM para cadastro de clientes e a criação de títulos do Agronegócio como CPR Física e Financeira.

**Observações**   
Esse sistema é apenas um protótipo.

## Detalhes do Sistema

No sistema é possível realizar o cadastro completo do cliente.

```bash
Informações Cadastrais
```
Informações básicas do cliente.

```bash
Informações do Cônjuge
```
Informações básicas do cônjuge.

```bash
Informações de Endereço
```
Informações básicas do endereço (consulta de CEP usando a API do ViaCEP.

```bash
Informações de Contato
```
Telefone e e-mail.

```bash
Outras Informações
```
